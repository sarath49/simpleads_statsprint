*********************************
	SimpleAds Stats Print
*********************************
Core 7.x

Allows printing of SimpleAds stats in pdf format.

Requirements:
---------------------------
SimpleAds module

Instruction:
----------------------------
Enable the module and go to Ad statitics page.
In this statistics page you will see Generate PDF link.
Clicking on link will genrate the pdf.

If you need to modify the color schme check simpleads-statsprint.tpl.php
file and override it.
